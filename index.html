Real Time Root Login Alert System

CLOUDTRAIL → EVENTBRIDGE → SNS

A serverless tripwire around the most powerful login in any cloud account. CloudTrail records every root login, an EventBridge rule matches the event the moment it lands, and SNS pushes an email alert straight to my inbox within seconds. No polling, no dashboards, no manual checks.

  • Implements a core CIS benchmark security control: alert on any root account use
  • Fully serverless and event driven, nothing to patch and effectively zero cost at idle
  • The same rule and target pattern extends to any account activity worth watching

Secure Global Document Delivery Pipeline

S3 → CLOUDFRONT · S3 VERSIONING · AWS BUDGETS

A locked down CDN for documents. Files live in a private S3 bucket with every form of public access blocked, and CloudFront serves them worldwide through Origin Access Control, so the edge is the only door in. S3 Versioning protects each file against overwrites and deletion, while AWS Budgets holds the whole pipeline under one cent a month.

  • Zero public buckets. The origin is reachable only through the CDN
  • Point in time recovery for every file via S3 Versioning
  • Global edge delivery for less than $0.01 a month, enforced by a budget alarm
